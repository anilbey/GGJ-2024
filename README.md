## Global Game Jam 2024

**Aman Doktor**

> "Aman doktor, canım gülüm doktor. Derdime bir çare. Çaresiz dertlere düştüm, doktor bana bir çare."

Amansız hastalıklarla tutulmuş vatandaşların dertlerine derman olma simülasyonu.
